Sistemas embebidos

Herramientas de aprendizaje y prototipado

Texas instruments
Los launchpads y booster packs de la línea de desarrollo de Texas Instruments, MSP430, Tiva C, C2000, Hercules. Microcontroladores de bajo consumo, ARM, procesadores para control en tiempo real.
DPScope
Herramientas de medición de calidad. Para trabajo en campo, realización de laboratorios en espacios reducidos. Adecuado para quienes desean hacer prácticas de electrónica y desean medir sus resultados. Osciloscopios digitales, analizadores de señales perfectos para las aplicaciones más comunes, para protocolos RS232, RS485, I2C y similares
Impresión 3D
Trabajamos impresión 3D en PLA, ideal para desarrollos de partes para robótica, accesorios para sistemas embebidos, diseño de actuadores, coleccionables, así como proyectos para diseñadores, partes de maquetas o el proyecto que necesites. 3D Print
Odroid
Tarjetas SBC Odroid, sistemas embebidos para el desarrollo de aplicaciones en Ubuntu o Android
Raspberry PI
Las conocidas plataformas de desarrollo Raspberry, encontrarás la B+, 2 y A+
Módulos y sensores
Módulos para pruebas, sensores y pequeñas tarjetas de expansión, todo para el desarrollo de prototipos electrónicos. Diseñamos y fabricamos tus prototipos.
Arduino
Tarjetas arduino, shields y módulos para Arduino.
Tutoriales
No solo encontraras las herramientas para tus trabajos, también códigos de ejemplo, tutoriales y pequeños cursos para quienes hasta ahora inician en la electrónica
