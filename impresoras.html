Impresoras y sistemas POS Custom

Puntos de venta - Sistemas POS

Custom® ha alcanzado altos niveles de especialización en la fabricación de sistemas de automatización de puntos de venta. Incluye soluciones inovadoras de equipos para la gestión de ventas, impresión de tiquetes y registro de ventas. Sistemas POS elegantes y versátiles.


Impresoras térmicas industriales y POS

Provee soluciones para diferentes tipos de aplicaciones, impresoras tipo panel, mecanismos de impresión, impresoras de adhesivos, fáciles de integrar a equipos de medición o control. Impresoras para quioscos o autoservicio. Impresoras reconocidas como la PLUS2 la cual permite su montaje en panel, rollos de papel más amplios e impresión de etiquetas: VKP80III sistema anti halado el cual garantiza mayor duración del cabezal de impresión y la cuchilla. TG2460H - TG2480H, impresoras industriales para trabajo pesado, de gran duración, con sistemas de protección patentados.

Disponibles impresoras para la industria de las estaciones de servicio, impresoras compatibles con combustibles y el ambiente hostil de las EDS, diseñadas para durar, consúltenos para mayor información.
Impresoras térmicas portátiles

Impresoras Bluetooth y WiFi, especiales para soluciones portátiles, sistemas de valet parking, ventas móviles, registro en punto, etc.

Facilita la impresión de boletos para eventos, tiquetes de productos, códigos de barras para facilitar el seguimiento de productos. Soluciones de impresión móvil, generación de boletos para eventos directamente el el punto. Custom® también provee soluciones para la industria de los juegos de azar, loteria y similares.
